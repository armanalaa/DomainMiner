 # TPC-H — Business Workflows

## 1. Query Validation Preparation

Performed By: Test Sponsor, Database Administrators

Activity:
- Build a qualification database in the same manner as the test database.
- Execute query validation tests using the provided query text and default substitution parameters.
- Run queries against the qualification database without any update activities.
- Ensure the resulting output matches the output data specified for each query.

## 2. Query Definition and Validation

Performed By: Database Analysts, Quality Assurance Engineers

Activity:
- Define functional query specifications based on business questions.
- Generate executable query texts using substitution parameters.
- Validate queries against the qualification database to ensure compliance with benchmark requirements.

## 3. Query Execution and Reporting

Performed By: Database Administrators, Data Analysts

Activity:
- Execute queries on the test database using the specified driver and implementation.
- Record results for each query in the full disclosure report.

## 4. Performance Test Preparation

Performed By: System Under Test Developers, Database Administrators

Activity:
- Ensure that no aspects of the System Under Test differ between this demonstration of compliance and the performance test.

## 5. Query Execution and Result Analysis (Performance Test)

Performed By: System Under Test Developers, Data Analysts

Activity:
- Execute queries on the test database to measure performance.
- Analyze results to determine if the query texts execute in accordance with the benchmark requirements.

## 6. Additional Assurance Requests (Optional)

Performed By: Auditors, System Under Test Developers

Activity:
- Request additional assurance that the query texts execute in accordance with the benchmark requirements.

## 7. Hardware Configuration Modifications (Optional)

Performed By: Database Administrators, System Under Test Developers

Activity:
- Build the qualification database on additional disks if necessary, as long as this change has no impact on the results of the demonstration of compliance.
