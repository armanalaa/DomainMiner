 # TPC-H — Business Workflows

## 1. Query Execution Management

Performed By: TPC-H Test Administrators

Activity:
- Define the order of query execution within a query set based on the ordering number (O(s)).
- Submit queries to the system in the defined sequence for execution.
- Monitor the progress of each query and ensure they are executed in an atomic fashion.

## 2. Query Stream Initiation

Performed By: TPC-H Test Administrators

Activity:
- Determine the first query stream that starts submitting queries before any other.
- Begin timing the measurement interval (T) when the first character of the executable query text is submitted to the SUT by the driver.

## 3. Query Execution

Performed By: System Users (Database Administrators, Analysts)

Activity:
- Execute TPC-H queries on various tables (customer, lineitem, nation, orders, part, partsupp, region, supplier) to analyze business data.
- Record the timing interval (QI(i,s)) for each query execution.

## 4. Refresh Function Execution

Performed By: System Users (Database Administrators, Analysts)

Activity:
- Execute refresh functions on specific tables (customer, lineitem, nation, part, supplier) to maintain data consistency and accuracy.
- Record the timing interval (RI(j,s)) for each refresh function execution.

## 5. Query Output Collection

Performed By: TPC-H Test Administrators

Activity:
- Collect output data from the last query of the last query stream.
- End the measurement interval (T) when the last character of output data is received by the driver from the SUT.

## 6. Query Timing and Reporting

Performed By: TPC-H Test Administrators

Activity:
- Round timing intervals to the nearest 0.01 second for reporting purposes.
- Calculate throughput based on the measurement interval (T) and the number of queries executed.

## 7. Query Set Sequencing

Performed By: TPC-H Test Administrators

Activity:
- In cases where the list of ordered sets in Appendix A is exhausted, follow the last reference with the first reference to continue query set execution.

## 8. System Usage Monitoring

Performed By: TPC-H Test Administrators

Activity:
- Monitor the system's real-time clock to ensure it has a resolution of at least 0.001 second for accurate timing intervals.
