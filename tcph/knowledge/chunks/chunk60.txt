 # TPC-H — Business Workflows

## 1. Query Generation and Execution

Performed By: Data Analysts, Database Administrators

Activity:
- Data analysts formulate queries based on business needs, using the provided schema.
- Database administrators generate executable query text for each query.
- Data analysts input substitution parameters for each query.
- Database administrators replace text-tokens by the query stream ID and submit complete executable queries to the system under test (SUT).
- Database administrators measure execution times of the queries and compute measurement statistics.
- Database administrators maintain an audit log of query text and query execution output.

## 2. Query Sequencing and Scheduling

Performed By: Data Analysts, Database Administrators

Activity:
- Data analysts sequence queries for execution by the query streams.
- Database administrators activate, schedule, and synchronize the execution of refresh functions in the refresh stream.

## 3. Refresh Function Execution

Performed By: Database Administrators

Activity:
- Database administrators submit each executable refresh function to the SUT for execution.

## 4. Query Result Processing and Analysis

Performed By: Data Analysts

Activity:
- Data analysts receive the output data resulting from each query execution from the SUT.
- Data analysts analyze the results to gain insights and make informed decisions.

## 5. System Under Test (SUT) Interaction

Performed By: Database Administrators

Activity:
- Database administrators interact with the SUT to execute queries and refresh functions.

## 6. Query Stream ID Generation

Performed By: Data Analysts, Database Administrators

Activity:
- Data analysts or database administrators generate a unique stream ID for each query stream, starting with 1 (or 0 for the power test).

## 7. Delay Management

Performed By: Database Administrators

Activity:
- Database administrators ensure delays between consecutive queries of the same query stream do not exceed half a second.

## 8. Query Text Modification Prohibition

Performed By: Data Analysts, Database Administrators

Activity:
- Data analysts and database administrators do not modify the compliant executable query text prior to its submission to the SUT.
