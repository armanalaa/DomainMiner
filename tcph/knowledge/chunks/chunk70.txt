 # TPC-H — Business Workflows

## 1. System Configuration and Setup

Performed By: Database Administrators, System Engineers

Activity:
- Database administrators configure the system with specified memory size for the test database.
- System engineers set up the SUT (System Under Test) according to the TPC-H standard specifications.
- Database administrators implement power test steps, including system boot and database restart.

## 2. Data Loading

Performed By: Database Administrators

Activity:
- Database administrators load the test data into the TPC-H database following the specified schema.

## 3. Performance Test Execution

Performed By: System Testers, Database Administrators

Activity:
- System testers execute the performance test on the SUT according to the TPC-H standard specifications.
- Database administrators record system activity logs between the load test and the beginning of the performance test.

## 4. Power Test Reporting

Performed By: Database Administrators, Testers

Activity:
- Database administrators report the timing intervals for each query and refresh function during the power test.
- Testers compile and archive the output for each query and refresh function from the power test.

## 5. Throughput Test Execution

Performed By: System Testers, Database Administrators

Activity:
- System testers execute the throughput test on the SUT according to the TPC-H standard specifications.
- Database administrators record start and finish times for each query stream during the throughput test.

## 6. Throughput Test Reporting

Performed By: Database Administrators, Testers

Activity:
- Database administrators report the number of query streams used in the throughput test.
- Testers compile and archive the output for each query stream from the throughput test.

## 7. Performance Metric Calculation

Performed By: Database Administrators, Testers

Activity:
- Database administrators calculate the memory/database size percentage according to the TPC-H standard specifications.
- Testers compute and report the performance metric (QphH@Size) and related numerical quantities from both runs.

## 8. System Activity Reporting (Between Runs)

Performed By: Database Administrators, Testers

Activity:
- Database administrators and testers document any system activity that takes place between the conclusion of Run1 and the beginning of Run2.

## 9. Query Output Validation Test

Performed By: Testers

Activity:
- Testers execute the Query Output Validation Test and report its output in the supporting files archive.

## 10. Documentation Provision

Performed By: Database Administrators, Testers

Activity:
- Database administrators and testers provide all necessary documentation to satisfy TPC-H standard specifications upon request.
